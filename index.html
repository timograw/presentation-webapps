REST

Representational State Transfer

Combination of HTTP & Services
(in our case)

A Resource or part of resource can be represented by an URL

List of shipments (Collection)http://hellmann.com/shipments/
Specific shipmenthttp://hellmann.com/shipment/12345
Destination agenthttp://hellmann.com/shipment/12345#destinationagent

Use HTTP Methods to GET, PUT, POST, DELETE Resources

Resources are serialized in XML or JSON

Same Origin Policy

Javascript can only access URLs that have the same domain

http://www.hellmann.com/test.js 
$.get("http://www.hellmann.com/data.json"); // Success 
$.get("data.json"); // Success 
$.get("http://www.google.com"); // Error
